# 🚀 Linguagem de Programação I - Informática para Negócios

Este repositório reúne os exercícios práticos desenvolvidos na disciplina de **Linguagem de Programação I** da FATEC São Bernardo do Campo. O foco é o aprendizado de lógica e algoritmos através de múltiplas linguagens, essenciais para a formação técnica no curso de **Informática para Negócios**.

---

## 🧰 Linguagens Utilizadas

O repositório contém resoluções de listas de exercícios em três linguagens principais:

* **Java:** Práticas com foco em tipagem e estrutura de objetos.
* **Python:** Exercícios focados em sintaxe limpa e produtividade.
* **C++:** Lógica de programação com foco em performance e manipulação de dados.

---

## 📂 Conteúdo dos Exercícios

As atividades cobrem os pilares fundamentais da programação:

1.  **Lógica Sequencial:** Operações aritméticas e manipulação de variáveis.
2.  **Estruturas Decisórias:** Implementação de fluxos de regras de negócio (If/Else, Switch-Case).
3.  **Laços de Repetição:** Automação de tarefas e processamento de dados (For, While).
4.  **Estruturas de Dados:** Uso de vetores e matrizes para organização de informações.
